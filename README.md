üí° The Problem & Solution
The Problem: Solo professionals (tutors, photographers, consultants) run their businesses on WhatsApp.

Leads get buried under personal chats.

"I'll pay you next week" promises are forgotten.

There is no "Dashboard" to see how much money is pending.

The Solution: A "Sidecar" extension for WhatsApp Web.

Snap: Extracts deal details (Name, Status, Money) from chat text using AI.

Save: Stores the lead in a cloud database (Supabase).

Track: Shows a "Pipeline View" of all active deals and total potential revenue.

üöÄ Quick Start Guide
1. Clone & Install
Bash

git clone https://github.com/yourusername/whatsapp-micro-crm.git
cd whatsapp-micro-crm
npm install
# or
pnpm install
2. Environment Setup
Create a file named .env.local in the root directory:

Code snippet

# Get this from platform.openai.com
PLASMO_PUBLIC_OPENAI_KEY=sk-proj-your-openai-key-here

# Get these from supabase.com
PLASMO_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
PLASMO_PUBLIC_SUPABASE_KEY=your-anon-key-here
3. Database Setup (Supabase)
Go to your Supabase SQL Editor and run this query to create the table:

SQL

create table leads (
  id bigint generated by default as identity primary key,
  created_at timestamp with time zone default timezone('utc'::text, now()) not null,
  name text,
  summary text,
  status text, -- 'Hot Lead', 'Closed', 'Cold'
  chat_log text
);
Tip: Disable RLS (Row Level Security) temporarily for testing.

4. Run Locally
Bash

npm run dev
5. Load into Chrome
Open Chrome and navigate to chrome://extensions.

Enable Developer Mode (top right toggle).

Click Load Unpacked.

Select the build/chrome-mv3-dev folder inside your project.

Go to [suspicious link removed] and click the extension icon!
